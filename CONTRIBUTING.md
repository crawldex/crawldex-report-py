# Contributing

This repository is a generated public mirror for `github.com/crawldex/crawldex-report-py`. The canonical source lives in the CrawlDex monorepo.

Pull requests are welcome here. Maintainers review public PRs, port accepted changes upstream into the canonical monorepo, then regenerate this mirror.

Guidelines:

- Keep examples read-only unless a user has explicitly authorized a mutating action.
- Do not include credentials, cookies, screenshots, browser traces, private customer data, or raw evidence artifacts.
- Do not include CAPTCHA, MFA, anti-bot, paywall, or account-control bypass instructions.
- Keep package metadata pointed at `https://github.com/crawldex/crawldex-report-py`.
- For generated files, explain the source change rather than hand-editing mirror output.
